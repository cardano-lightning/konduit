## Architecture

We adopt [c4 model](https://c4model.com/abstractions) conventions.

```
Software System >> Containers >> Compontents >> Code
```

Where `>>` can be read as "is made of multiple".

### Containers

Internal:

- App
- Node
- User L1 Liaison

External:

- Cardano Node
- BLN Node
- Price feed source
- BLN Invoicer
- Mobile OS API
